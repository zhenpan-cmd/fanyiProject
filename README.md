# 单词记忆卡片应用

一个用于英语学习的交互式单词记忆卡片应用，支持多个词汇级别（四级、六级、雅思）。

## 功能特点

- **多级词汇**: 包含四级(CET4)、六级(CET6)和雅思(IELTS)三个级别的词汇
- **详细信息**: 每个单词都提供音标、中文释义、详细解释和例句
- **卡片翻转**: 点击按钮可翻转卡片查看单词翻译
- **即时查询**: 支持输入单词进行即时翻译查询
- **级别筛选**: 可根据学习需求选择不同的词汇级别

## 使用方法

1. 选择词汇级别（四级、六级或雅思）
2. 点击"下一个单词"浏览词汇
3. 点击"显示翻译"查看单词详细信息
4. 使用底部的翻译区域查询任意单词

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)

## 词汇级别

- **CET4**: 大学英语四级词汇，适合基础英语学习者
- **CET6**: 大学英语六级词汇，适合进阶英语学习者
- **IELTS**: 雅思考试词汇，适合准备雅思考试的学习者

每个单词都包含以下信息：
- 英文单词
- 音标
- 中文释义
- 详细解释
- 英文例句及其中文翻译
